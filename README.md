# Expense Tracker

A professional React-based expense tracker application with Indian currency support, dark/light mode and comprehensive transaction management.

## Features

✅ **Multi-Currency Support** - INR (Indian Rupee) as default, plus USD, EUR, GBP, JPY, AUD, CAD

✅ **Dark/Light Mode Toggle** - Seamless theme switching with localStorage persistence

✅ **Transaction Management** - Add, view, delete income and expense transactions

✅ **Smart Filtering** - Search by description, filter by type and category

✅ **Date Tracking** - Calendar date picker for each transaction

✅ **Real-time Calculations** - Auto-calculated balance, total income, and total expenses

✅ **Categories** - Pre-defined categories for income and expenses

✅ **Local Storage** - All data persists in browser localStorage

✅ **Professional UI** - Modern minimal design with Shadcn UI components

✅ **Responsive Design** - Works perfectly on desktop, tablet, and mobile
